**Entertainment Games Soft (EGS)** is a non-typical private gambling development studio intended to be established in Macau. We believe the future of the industry lies not in the "house edge" but in **technological transparency** and **the aesthetics of pure probability**.

* **Non-Profit Oriented**: Our mission is to explore the essence of gambling behavior through code and mathematical research.
* **Full-Stack Open Source**: From RNG (Random Number Generators) to backend engines, all source code is publicly available on GitHub for audit.

## üß™ Research Focus

### 1. Discrete Probability & Fair Algorithms
We are dedicated to achieving true **Provably Fair** systems. 
By utilizing the $\text{HMAC-SHA256}$ algorithm, we ensure that every game result is pre-determined and immutable before it is even generated.

$$P(G) = \frac{1}{\sum_{i=1}^{n} w_i}$$

### 2. Large Language Models (LLM) & Gambling Psychology
We are researching how to leverage LLMs for:
* **Dynamic Narrative Generation**: Adjusting visual storylines in real-time based on a player's betting style.
* **Responsible Gambling Alerts**: AI-driven semantic warnings triggered when irrational "loss-chasing" behavior is detected.
